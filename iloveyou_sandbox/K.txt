The simulation code is **embedded directly in this README** (see [📄 Complete Simulation Code](#-complete-simulation-code)). Copy it into a `.py` file to run it.

**Key functions inside the script:**
- `prepare_sandbox()` – Creates the test folder and dummy files.
- `worm_self_copy()` – Reads its own source code (the payload to be written).
- `overwrite_files()` – Walks through the sandbox and replaces matching files.
- `simulate_email_propagation()` – Prints fake Outlook emails.
- `simulate_password_steal()` – Prints dummy stolen credentials.
- `add_to_registry()` – Prints the startup registry key.

All functions are clearly commented so you can follow the logic step by step.
